Detailed Description

template<class Line_format>
class ESBTL::Atom_list_occupancy_policy< Line_format >

Keep only a set of atoms given by their atom serial numberThis object provides an occupancy policy to keep atoms with an occupancy != 1 and with no alternate location using their atom serial number.

Template Parameters
Line_formatis a helper class to read molecular data files such as ESBTL::PDB::Line_format.
